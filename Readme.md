Matru-Sneh Health 🌸

Matru-Sneh Health is an Android-based pregnancy healthcare application developed using modern Android technologies and GenAI-assisted development concepts. The application is designed to help pregnant women maintain healthy daily routines through smart healthcare monitoring, reminders, hydration tracking, and personalized health alerts.

The application provides a simple, responsive, and user-friendly healthcare platform that supports maternal wellness through intelligent mobile features.

 Features

✅ Smart Reminders

Morning hydration reminders

Afternoon nutrition reminders

Night rest and sleep reminders

Time-based notifications using WorkManager


✅ Health Alerts

Water intake alerts

Pregnancy vitamin reminders

Rest suggestions

Daily healthcare suggestions

Toggle ON/OFF functionality


✅ Water Intake Tracker

Track daily water consumption

Increase/decrease water count

Hydration progress monitoring

Daily monitoring support
